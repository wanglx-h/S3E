Subject: Odd number
An '''odd number''' is an integer when divided by two, either leaves a remainder or the result is a Fraction (mathematics). One is the first odd positive number. Some examples of odd numbers are 1, 3, 5, 7, 9, and 11. An integer that is not an odd number is an even number. If an even number is divided by two, the result is another integer. On the other hand, an odd number, when divided by two, will result in a fraction.
