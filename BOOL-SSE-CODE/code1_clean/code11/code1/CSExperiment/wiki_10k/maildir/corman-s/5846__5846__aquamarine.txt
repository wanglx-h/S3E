Subject: Aquamarine
Image:Aquamarin cut.jpg '''Aquamarine''' is a gemstone, named for its blue-green color, which looks like some shallow seas. It is a type of beryl. The color aquamarine (color) is named for the gemstone. It is considered as a birthstone of March. ==Other websites== Category:Gemstones Category:Birthstones Category:Beryl
