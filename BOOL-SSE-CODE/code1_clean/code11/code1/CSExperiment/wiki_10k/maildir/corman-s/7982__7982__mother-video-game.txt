|genre = Role-playing game |modes = Singe player |platforms = Famicom, Nintendo Entertainment System (cancelled) }} '''''Mother''''' (マザー Mazā) is a 1989 video game sold only in Japan for the Nintendo Entertainment System (FC). They planned to sell the game in the United States, but Nintendo chose not to release it. This is because when they finished making it into English, the Super Nintendo Entertainment System was already being sold.
