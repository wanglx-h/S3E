Subject: Landjäger
Image:Landjäger.jpg '''Landjäger''' (or '''Landjaeger''') is a type of Germany sausage. The name stands for "hunter's sausage". It is smoked, so it does not need refrigerator. Usually, some of the water has also been taken out (they are dehydration). Category:Sausage
