The '''MediaWiki namespace''' is a way to change the look of Wikipedia. You can change the words and text used for site links and text. For example, the "" message which you can see at the top of each page is available at MediaWiki:Tagline. Only Wikipedia:Administrators can change the MediaWiki namespace. If you want to make changes, please discuss them on Wikipedia talk:MediaWiki namespace.
