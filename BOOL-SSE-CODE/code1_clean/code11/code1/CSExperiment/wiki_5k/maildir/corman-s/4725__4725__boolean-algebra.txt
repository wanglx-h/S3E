Subject: Boolean algebra
In mathematics, '''Boolean algebra''' is an algebra for Binary number digits (where 0 means false and 1 means true). It is equipped with three Operation (mathematics): conjunction (AND), disjunction (OR) and negation (NOT). It uses normal math symbols, but it does not work in the same way. It is named for George Boole, who invented it in the middle 19th century. Boolean algebra did not get much attention except from mathematicians until the 20th century when engineers began using it for logic gates. == NOT gate == {| class="wikitable" |- !
