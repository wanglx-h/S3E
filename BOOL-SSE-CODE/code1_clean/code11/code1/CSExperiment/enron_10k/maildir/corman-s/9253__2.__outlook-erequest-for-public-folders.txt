Subject: Outlook -eRequest for Public Folders
In order to help expedite the setup of your Public Folder, please complete an eRequest. If you have questions regarding public folders, please contact Debra Kelly at extension 3-4269. 1. Open up <http://itcapps.corp.enron.com/srrs/entry/srrs.asp> and choose Corp Logon and log in with your regular network credentials. 2. Click the triangle for Update Account Access. 3. Type the email address of the person who the request is for and click OK. 4. Click the Applications circle button. 5. In the text search box, type in "Outlook" and click Search. 6. Click the Add button next to Outlook. 7. Check the Public Folder check box and click Add to Request 8. Enter the required information and click Continue. The information we MUST have for each Public Folder request is: ? Exact name of the Public Folder (if it is a person's name, it can be in the form of first and last name or first initial last name, like Joe User or JUser) ? Exact location of the folder such as Global Functions/IT/Messaging/JUser [For users on the 38th floor the folder should be located under: All Public Folders; Corporate; Legal; ENA Legal.] ? Who needs access to this folder (it can be limited just to one person or several people can have access) 9. Verify the request information and click Submit.
