Subject: Web browser
A '''web browser''' is a computer program application for reading pages of the World Wide Web. Since the late 1990s, most personal computer and Mobile phone and other mobile_device have a browser. Web browsers are used by people to find and look at Website on the Internet. The first web browser was created in 1990. Many web browsers are available for free. All web browsers can go to websites but each browser has good things and bad things about it. For example, some browsers focus on Information security and keeping computers safe from computer virus. Other browsers are made so that web pages appear on-screen faster.
